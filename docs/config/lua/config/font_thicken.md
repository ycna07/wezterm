# `font_thicken = false`

Draw fonts with a thicker stroke.

This is implemented for the `FreeType` font rasterizer and applies to
outline glyphs.

```lua
config.font_thicken = true
```

Use [font_thicken_strength](font_thicken_strength.md) to control the amount
of thickening.
